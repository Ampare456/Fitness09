# Fitness-09
ระบบสมาชิกฟิตเนส **IRONCLUB** พร้อมใช้งานแล้ว! 💪

---

## ฟีเจอร์ทั้งหมด

**📊 Dashboard** — สถิติรวม, เช็คอินวันนี้, สมาชิก active/หมดอายุ

**👥 จัดการสมาชิก**
- เพิ่ม/แก้ไข/ลบสมาชิก
- เก็บข้อมูล: ชื่อ, เบอร์, อีเมล, เพศ, วันเกิด, หมายเหตุ
- แพ็กเกจ: รายเดือน / 3 เดือน / 6 เดือน / รายปี / รายครั้ง
- แสดงสถานะ Active / หมดอายุ

**🔲 QR Code**
- สร้าง QR ให้สมาชิกแต่ละคนอัตโนมัติ
- กดพิมพ์ได้เลย

**✅ Check-in**
- กรอกรหัสหรือคลิกจากรายชื่อ
- เช็คแพ็กเกจหมดอายุทันที
- ล็อกประวัติทุกครั้ง

**💾 บันทึกข้อมูล** — ใช้ localStorage เปิดซ้ำไม่หาย
